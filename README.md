# MERN Product Management Application

## Features

* User Registration
* User Login
* Add Product
* Edit Product
* Like / Unlike Product
* Liked Products
* User Profile
* Logout

## Tech Stack

* React.js
* Node.js
* Express.js
* MongoDB
* JWT Authentication

## Installation

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

Create a `.env` file in the backend folder:

```env
PORT=3000
MONGO_URI=YOUR_MONGODB_URI
JWT_SECRET=YOUR_SECRET_KEY
```
